GUNPONG

Gunpong is a simple multiplayer game that can be visited in a browser and played with the arrow keys. The objective of the game is to keep the ball from escaping a small circle with a paddle that follows a rotary track around a circle. Many players join this game and are able to participate with the real-time syncing made possible with Gun.
